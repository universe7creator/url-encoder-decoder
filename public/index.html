FAQ

What's the difference between encode and encodeComponent?

encodeURI encodes the full URL but preserves special characters like : / ? # that are valid in URLs. encodeURIComponent encodes everything including those characters, making it safe for query parameter values.

Can I process multiple URLs at once?

Yes! The Pro version includes batch processing capabilities. Paste multiple URLs (one per line) and process them all at once.

Is my data secure?
